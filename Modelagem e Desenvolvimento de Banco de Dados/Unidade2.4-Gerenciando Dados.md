Tendo como base a figura abaixo, são expostas algumas entidades e relacionamentos no contexto de uma determinada universidade.

Imagine que você é um funcionário dessa universidade e precisa extrair informações de gestão desses relacionamentos. Quais informações poderiam ser utilizadas em um processo de tomada decisão, como, por exemplo:  
1- Qual é o professor de determinada disciplina?  
2- Quantos alunos estão matriculados em determinada disciplina?  
Cite pelo menos mais cinco informações úteis para a gestão dessa universidade.



####  ✅ Resposta ao desafio:

A partir das entidades e relacionamentos do Banco de Dados, é possível extrair várias informações estratégicas como exemplo:
- Professor responsável: identificar qual docente ministra determinada disciplina.
- Número de alunos: verificar quantos estudantes estão matriculados em cada curso ou disciplina.
- Taxa de evasão: acompanhar quantos alunos cancelaram ou não concluíram a matrícula em um curso.
- Distribuição de carga horária: analisar se os professores estão sobrecarregados ou se há equilíbrio na atribuição de disciplinas.
- Notas e desempenho: avaliar o rendimento dos alunos em cada disciplina para identificar pontos críticos.
- Oferta de cursos: verificar quais cursos estão sendo mais procurados e quais têm baixa demanda.
- Horários de disciplinas: identificar conflitos de horários que dificultam a matrícula dos alunos.
- Histórico de matrículas: acompanhar a evolução da quantidade de alunos matriculados ao longo dos semestres.

Essas informações permitem decisões como: contratar mais professores, abrir novas turmas, ajustar horários, criar políticas de retenção de alunos e até revisar a grade curricular.

#### Questão 1
Mesmo não estando ciente, as tecnologias de banco de dados afetam muito sua vida. Cabe citar também que os bancos de dados computadorizados são de vital importância para as organizações modernas. Nesse contexto, selecione abaixo uma opção em que NÃO estão envolvidas tecnologias computadorizadas de armazenamento de dados.

Selecione a resposta:
a. Realizar compras e fazer pagamento no caixa do supermercado.
b. Realizar um depósito ou saque em um caixa eletrônico.
c. Efetuar a matrícula online em um curso de línguas.
d. Fazer a leitura de um capítulo de um livro.
e. Realizar uma aposta em uma casa lotérica.

#### Questão 2
Todos os dias, pessoas e empresas coletam inúmeros dados, como nomes, valores, RG, CPF, contas bancárias, endereços, números de cartões de crédito, entre outros. Para manter esses dados disponíveis por longo tempo, os bancos de dados utilizam uma característica técnica, a qual pode ser definida como persistência. Nesse contexto, selecione abaixo a definição que melhor define o termo persistência:

Selecione a resposta:
a. É uma característica que define que todas as ações que compõem a unidade de trabalho da transação devem ser concluídas com sucesso, para que seja efetivada. Se durante a transação qualquer ação que constitui unidade de trabalho falhar, a transação inteira deve ser desfeita (rollback).

b. É uma característica de um estado que sobrevive ao processo que o criou. No escopo da computação e banco de dados, conseguimos atingir esta característica através do armazenamento não volátil (disco rígido, memória flash, etc).

c. É uma característica que define que todas as regras e restrições definidas no banco de dados devem ser obedecidas. Relacionamentos por chaves estrangeiras, checagem de valores para campos restritos ou únicos devem ser obedecidos para que uma transação possa ser completada com sucesso.

d. É uma característica que define que cada transação funciona completamente à parte de outras estações. Todas as operações são parte de uma transação única. O princípio é que nenhuma outra transação, operando no mesmo sistema, possa interferir no funcionamento da transação corrente (é um mecanismo de controle).

e. É uma característica técnica que significa que os resultados de uma transação são permanentes e podem ser desfeitos somente por uma transação subsequente.

✅ A alternativa correta é: b. É uma característica de um estado que sobrevive ao processo que o criou. 

#### Questão 3

Nesse contexto, selecione abaixo uma opção em que se identifica um inter-relacionamento de dados.

Selecione a resposta:
a. Em determinada operação, os seguintes dados são relacionados: nome do cliente X número do pedido.

b. Em determinada operação, os seguintes dados são relacionados: nome do cliente X endereço do cliente.

c. Em determinada operação, os seguintes dados são relacionados: nome do cliente X CPF do cliente.

d. Em determinada operação, os seguintes dados são relacionados: endereço do cliente X RG do cliente.

e. Em determinada operação, os seguintes dados são relacionados: data de nascimento do cliente X filiação do cliente.

✅ A alternativa correta é: a. nome do cliente X número do pedido.

As transações são elementos-chave muito importantes em SGBDs. Podemos definir uma transação como:

## Na prática

O gerenciamento de dados nos dias de hoje

Um banco de dados é uma coleção de dados armazenados e organizados de modo a atender as necessidades integradas dos seus usuários. Possibilita a consulta e a manipulação dos dados e pode ser manual ou computadorizado.

Como exemplo de banco de dados manual, pode-se citar o prontuário médico em registros de papel, armazenados em pastas, as quais são organizadas em ordem alfabética, a partir do nome do paciente.
